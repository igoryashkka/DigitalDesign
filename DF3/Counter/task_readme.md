# DF3 / Counter Task

## Що треба зробити
Завершити FPGA-проєкт лічильника з кнопками, PWM-виходом і перемиканням режиму через перемикач.

## Піни / порти
- `top.vhd`
  - `clk`, `rst_n` - вхід синхронізації та скидання
  - `sw_op` - вибір джерела значення
  - `btn_a_up`, `btn_a_down`, `btn_b_up`, `btn_b_down` - кнопки керування
  - `pwm_muxed_o` - PWM-вихід
  - `led(3 downto 0)` - індикація молодших бітів

## Що перевірити
- Дебаунс кнопок і формування одного імпульсу на натискання.
- `updown_byte` має коректно змінювати значення в межах розрядності.
- PWM-вихід повинен відповідати значенню, яке обрано через `sw_op`.
