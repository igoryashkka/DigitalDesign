# DV2 / FilterDXI Task

## Що треба зробити
Побудувати та перевірити VHDL-фільтр з обробкою 3x3 вікна для різних ядер: Laplacian, Gaussian і averaging.

## Піни / порти
- `dxi_top.vhd`
  - `i_clk`, `i_rstn` - вхід синхронізації та скидання
  - `i_dxi_valid`, `i_dxi_data(71 downto 0)` - вхідний DXI-трансфер
  - `o_dxi_ready` - готовність приймати новий блок даних
  - `i_dxi_out_ready` - готовність приймача на виході
  - `o_dxi_out_valid` - валідність вихідного результату
  - `o_master_data(7 downto 0)` - вихідний піксель/результат
  - `config_select(1 downto 0)` - вибір фільтра

## Що перевірити
- Обробку всіх чотирьох режимів `config_select`.
- Обмеження результату в діапазоні 0..255.
- Узгодження handshake між `valid/ready` на вході та виході.
