# DF3 / Decoder Task

## Що треба зробити
Зібрати декодерний FPGA-проєкт: арифметика, дисплей, UART-частина і керування через кнопки та зовнішні інтерфейси.

## Піни / порти
- `top.vhd`
  - `clk`, `rst_n` - входи
  - `btn_raw(5 downto 0)` - сирі кнопки
  - `led_zero_o`, `led_carry_o`, `led_over_o`, `led_neg_o` - прапори ALU
  - `pwm_r_o`, `pwm_g_o`, `pwm_b_o` - PWM для RGB
  - `dsp_used_o` - індикатор використання DSP
  - `sda_in`, `sda_out_en`, `sda_out`, `scl` - лінії дисплея / I2C-подібного інтерфейсу
  - `dsp_use_i`, `dsp_result_i(N_PWM_BITS-1 downto 0)`, `dsp_a_val`, `dsp_b_val` - інтерфейс до DSP-блоку

## Що перевірити
- Кнопки мають змінювати операнди та режим без дребезгу.
- ALU-прапори мають відповідати реальному результату операції.
- Дисплей та PWM мають отримувати дані з одного джерела результату без розсинхронізації.
