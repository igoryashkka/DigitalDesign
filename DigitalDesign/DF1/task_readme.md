# DF1 Task

## Що треба зробити
Зібрати й перевірити базові VHDL-компоненти для цифрової схемотехніки: інвертор, RS/D-защіпки, повний суматор, мультиплексор і просту FSM для світлофора.

## Піни / порти
- `inv.vhd`
  - `a(3 downto 0)`, `b(3 downto 0)` - входи
  - `y1`..`y5`, `y(3 downto 0)` - виходи
- `d-latch.vhd`
  - `clk`, `d` - входи
  - `q`, `qn` - виходи
- `fulladder.vhd`
  - `a(31 downto 0)`, `b(31 downto 0)`, `cin` - входи
  - `s(31 downto 0)`, `cout` - виходи
- `mux4to1.vhd`
  - `sel(1 downto 0)` - вибір каналу
  - `result` - вихід
- `traffic_fsm.vhd`
  - `clk`, `reset`, `Ta`, `Tb` - входи
  - `La1`, `La0`, `Lb1`, `Lb0` - виходи

## Що перевірити
- Після `reset` схема має перейти в безпечний стан.
- Для суматора перевірити `cin = 0/1` і перенос на старшому біті.
- Для `traffic_fsm` перевірити послідовність станів і реакцію на сенсори `Ta` та `Tb`.
