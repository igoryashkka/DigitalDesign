# Analog Mixed Verification

## AMS0

`AMS0` містить behavioral AMS-модель простого АЦП `adc_simple_001`.

Що робить модуль:

- приймає аналоговий вхід `IN`;
- квантує його відносно `VREF`;
- вибирає значення на фронті `CLK`;
- вимагає валідних `VDD3V3`, `VREF`, `IBIAS10U` та активного `EN`;
- віддає 8-бітний код через окремі виходи `OUT0..OUT7`;
- має затримку результату на один такт.

Склад каталогу:

- `AMS0/simple_adc.ams` - базова 8-бітна модель ADC;
- `AMS0/simple_adc_001.ams` - альтернативна параметризована версія;
